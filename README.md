# Power-BI-Data-Cleansing
Data Cleaning using Power BI.

## Objective
Clean and transform of Raw Data Set using Power Query.

## Data Cleaning Steps
- Use First Row as Headers
- Removed Duplicates
- Filtered Rows
- Replaced Values
- Add/Delete columns
- Removed and Merged Columns
- Split Delimeter
- Uppercase & Lowercase Text
- Rounded Up
- Capitalized Each Word
- Replaced Null values
  
*Note: Date field has been left as null, as it requires accurate info which can be provided by the Data Engineer.*
